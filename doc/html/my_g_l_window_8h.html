class  myGLWindow
 The myGLWindow class - Qt Widget for OpenGL Designed to support only one shape and one selection at time. So in the containers of the renderer there are two shapes at position 0 and 1. The first one for the current shape and the second one for the current selection. This object adapts a renderer to a QT OpenGL Widget which manages a single OpenGL context.
